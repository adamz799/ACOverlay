# AC Overlay - 神力科莎遥测数据显示插件

一个实时显示 Assetto Corsa (神力科莎) 遥测数据的透明悬浮窗口插件，支持数据录制和 RaceChrono 兼容格式导出。

## 功能特性

### 📊 实时数据监测
- 油门/刹车输入曲线图（历史轨迹）
- 油门/刹车垂直进度条
- 当前挡位和速度
- 转速指示灯（RPM）
- G力显示（横向/纵向）
- 方向盘转向角度

### 🎮 智能显隐
- 自动检测驾驶状态，开始驾驶时显示，停止时隐藏
- 手动控制可覆盖自动逻辑

### 📝 遥测数据录制 (新功能!)
- **自动录制**: 驾驶时自动开始录制
- **手动控制**: F11 手动开始/停止录制
- **RaceChrono 兼容**: CSV 格式可直接导入 RaceChrono 分析
- **文件命名**: `时间_赛道_车辆.csv` 格式，便于管理

### 记录的数据字段 (80+项)
| 类别 | 字段 |
|------|------|
| **时间** | 时间戳、圈数、圈时间 |
| **输入** | 油门、刹车、离合、方向盘角度 |
| **运动** | 速度(km/h, m/s)、挡位、转速 |
| **速度向量** | X/Y/Z 三轴速度 |
| **G力** | 横向、纵向、垂直加速度 |
| **姿态** | 航向角、俯仰角、横滚角 |
| **角速度** | X/Y/Z 三轴角速度 |
| **位置** | 赛道位置、行驶距离、世界坐标 |
| **轮胎** | 滑动、负载、胎压、温度、磨损 (4轮) |
| **刹车** | 刹车温度 (4轮) |
| **悬挂** | 悬挂行程、车身高度 |
| **燃油** | 剩余燃油、每圈油耗 |
| **动力** | 涡轮增压、ERS/KERS 能量 |
| **辅助** | TC、ABS、刹车前后比、发动机制动 |
| **DRS** | 可用状态、启用状态 |
| **损伤** | 前/后/左/右/中 损伤程度 |
| **状态** | 进站状态、限速器、扇区、出界轮胎数 |
| **环境** | 气温、路温、风速、风向 |
| **比赛** | 排名、旗帜状态 |

### ⌨️ 快捷控制
| 快捷键 | 功能 |
|--------|------|
| `F9` | 切换显示/隐藏 |
| `F10` | 切换自动显隐功能 |
| `F11` | 开始/停止录制 |
| `F12` | 切换自动录制功能 |

### 🖼️ 界面特性
- 无边框透明窗口，浮在游戏画面上层
- 支持拖拽移动位置
- 连接状态实时指示
- 录制状态指示（红色闪烁）
- 系统托盘菜单控制

## 安装

### 1. 安装 Python 3.10+

确保已安装 Python 3.10 或更高版本。

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 运行程序

```bash
python main.py
```

## 使用说明

### 基本使用
1. 启动 Assetto Corsa 游戏
2. 运行本程序
3. 程序会自动连接到游戏的共享内存接口
4. 连接状态指示灯：
   - 🟢 绿色 = 已连接
   - 🔴 红色 = 未连接/等待连接

### 数据录制
1. **自动录制** (默认开启): 检测到驾驶时自动开始录制
2. **手动录制**: 按 `F11` 开始/停止录制
3. **日志位置**: 程序目录下的 `logs` 文件夹
4. **文件格式**: `20240101_120000_spa_ferrari_488.csv`

### 系统托盘
- 右键点击托盘图标打开菜单
- 双击托盘图标切换显示状态
- 可快速访问日志文件夹

## 日志文件格式

生成的 CSV 文件兼容 RaceChrono，包含以下特性：
- 文件头包含会话元数据（车辆、赛道、时间等）
- 标准化的列名，可直接导入分析工具
- 高精度浮点数据（6位小数）
- 约 60Hz 采样率

### 示例输出
```csv
# AC Overlay Telemetry Log
# Session Start: 2024-01-01 12:00:00
# Car: ferrari_488_gt3
# Track: spa
# Player: John Doe
# Track Length: 7004.00 m
# Max RPM: 8500
#
Time (s),Lap #,Lap Time (ms),Speed (km/h),Speed (m/s),Throttle (%),Brake (%),...
0.000000,0,0,0.000000,0.000000,0.000000,0.000000,...
0.016667,0,16,5.234567,1.454324,45.678901,0.000000,...
```

## 在 RaceChrono 中使用

1. 将 CSV 文件传输到手机/平板
2. 打开 RaceChrono Pro
3. 选择 "导入会话"
4. 选择生成的 CSV 文件
5. 即可查看圈速分析、轨迹对比等

## 打包发布

生成独立可执行文件：

```bash
pip install pyinstaller
pyinstaller ACOverlay.spec --clean
```

生成的 `ACOverlay.exe` 位于 `dist` 目录。

## 技术说明

本程序通过读取 Assetto Corsa 的共享内存接口获取遥测数据：

- `acpmf_physics` - 物理数据（油门、刹车、速度、G力等）
- `acpmf_graphics` - 图形数据（游戏状态、位置等）
- `acpmf_static` - 静态数据（车辆、赛道信息等）

## 已知问题

- 需要管理员权限运行才能注册全局热键
- 仅支持 Windows 系统
- 需要先启动游戏再运行本程序，或等待自动重连
- 世界坐标基于游戏内部坐标系，非真实 GPS
